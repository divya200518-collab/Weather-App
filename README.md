# 🌦️ Java Weather App

A simple Java Weather Application that fetches real-time weather information using the OpenWeather API.

## Features

- Search weather by city
- Real-time temperature
- Humidity
- Wind speed
- Weather description
- Error handling
- Clean Java code

## Technologies

- Java
- OpenWeather API
- HTTPURLConnection
- Git
- GitHub

## How to Run

1. Clone the repository.
2. Add your OpenWeather API key.
3. Compile the Java files.
4. Run Main.java.

- GUI using JavaFX
